# Data Structures and Algorithms (DSA)

This repository contains exercises and implementations related to **Data Structures and Algorithms**.  
The purpose of this repository is to practice problem solving, understand algorithmic thinking, and implement fundamental programming techniques using Java.

---

## 1. Clone the GitHub Repository

Open the command prompt (Terminal) and navigate to the folder where you want to clone the project.

Run the following command:

git clone https://github.com/GezimC/DSA_2026.git

This will download the repository to your local machine.

---

## 1.1 Alternative – Download as ZIP

You can also download the repository as a **ZIP file** from GitHub.

Steps:
1. Open the repository page.
2. Click **Code**.
3. Select **Download ZIP**.

Note: Downloading as ZIP will only give you a snapshot of the current code and you will not be able to pull future updates from the repository.

---

## 2. Open the Project

You can open the project using any Java IDE, such as:

- IntelliJ IDEA
- Eclipse
- Visual Studio Code

After opening the project, navigate to the desired Java file and run it.

---

## 3. Running the Code

Each exercise contains a Java file with a `main` method.

You can run the code directly from your IDE.

Alternatively, you can run it from the command line:

javac FileName.java  
java FileName

Make sure Java is installed on your system.

---

## Recommendations

Always pull the latest changes before starting to work:

git pull origin main

Commit your changes frequently:

git add .  
git commit -m "My latest changes"  
git push origin main

If you want to experiment with code, it is recommended to create a **separate Java file** to avoid conflicts when pulling changes.

If you want to reset your local project and get a fresh copy of the repository, run:

git reset --hard origin/main

Warning: This command will discard all local changes.
